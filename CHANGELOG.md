# 更新日志 / Changelog

本项目遵循 [Semantic Versioning](https://semver.org/) 语义版本规范。

## [1.0.0] - 2026-03-17

### 🎉 首次发布

#### ✨ 新增功能

- **Agent 管理平台**
  - Agent 增删改查，支持自定义系统提示词
  - Skill（技能）配置与自动发现
  - MCP 工具源配置，支持在线探测可用工具
  - 基于 JWT 的用户认证与登录

- **智能对话**
  - 基于 DeepAgents 框架的 Agent 运行时
  - SSE 流式输出（思考过程、工具调用、最终回答）
  - 多轮对话支持，服务重启后自动恢复上下文
  - 对话历史持久化与自动加载

- **任务规划**
  - Agent 自动制定执行计划（write_todos）
  - 实时展示计划进度，支持动态调整
  - 前端左右分栏展示计划 / 执行详情

- **工具调用**
  - MCP 工具自动发现与调用
  - 工具调用重试机制（最多 3 次，失败跳过）
  - 工具调用展示中文名与函数名

- **报告生成**
  - 分析结果导出为 PDF 报告
  - PDF 支持中文字体与自定义水印
  - 前端一键下载

- **前端界面**
  - 博观品牌 UI，SVG 动态 Logo
  - 响应式布局，暗色主题
  - Agent 配置与对话双面板切换

#### 🏗️ 基础设施

- Docker / Docker Compose 部署支持
- `pyproject.toml` 标准化 Python 项目配置
- `.env` 环境变量管理
- CI / CD 基础模板（GitHub Actions）
